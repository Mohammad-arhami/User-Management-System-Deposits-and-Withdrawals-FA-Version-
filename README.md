## 📖 Overview / معرفی

### فارسی 🇮🇷

یک سیستم مدیریت کاربران مبتنی بر وب با قابلیت‌های پیشرفته برای مدیریت حساب‌های کاربری، واریز، برداشت و پیگیری موجودی از طریق رابط کاربری ساده و کارآمد. این پروژه مهارت‌های عملی در توسعه فرانت‌اند، مدیریت وضعیت و طراحی تعامل با کاربر را نشان می‌دهد. سیستم به مدیران امکان می‌دهد کاربران را ایجاد و مدیریت کنند، واریز و برداشت را با به‌روزرسانی لحظه‌ای موجودی پردازش کنند و فعالیت‌های مالی را به طور مؤثر نظارت نمایند. این پروژه با HTML5، CSS3 و جاوااسکریپت (ES6+) ساخته شده و از Local Storage برای ذخیره‌سازی داده استفاده می‌کند. رابط کاربری کاملاً واکنش‌گرا با داشبوردی تمیز و کاربرپسند است.

### English 🇬🇧

An advanced web-based user management system for handling user accounts, deposits, withdrawals, and balance tracking through an intuitive interface. This project demonstrates practical skills in front-end development, state management, and user interaction design. The system allows administrators to create and manage users, process deposits and withdrawals with real-time balance updates, and monitor financial activities efficiently. Built with HTML5, CSS3, and JavaScript (ES6+) using Local Storage for data persistence. Fully responsive with a clean dashboard layout.

## 🛠️ Installation / نصب و اجرا

### فارسی 🇮🇷

1. مخزن را کلون کنید:
   ```bash
   git clone https://github.com/Mohammad-arhami/User-Management-System-Deposits-and-Withdrawals-FA-Version-.git
   ```

2. به پوشه پروژه بروید:
   ```bash
   cd User-Management-System-Deposits-and-Withdrawals-FA-Version-
   ```

3. فایل `index.html` را در مرورگر باز کنید (یا از Live Server در VS Code استفاده کنید).

> به هیچ سرور یا فرآیند Build نیازی نیست. همه چیز مستقیماً در مرورگر اجرا می‌شود.

### English 🇬🇧

1. Clone the repository:
   ```bash
   git clone https://github.com/Mohammad-arhami/User-Management-System-Deposits-and-Withdrawals-FA-Version-.git
   ```

2. Navigate to the project folder:
   ```bash
   cd User-Management-System-Deposits-and-Withdrawals-FA-Version-
   ```

3. Open `index.html` in your browser (or use Live Server in VS Code).

> No server or build process required. Everything runs directly in your browser.
